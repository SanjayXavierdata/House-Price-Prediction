 House Price Prediction using Machine Learning

## 📌 Project Overview

This project predicts house prices using Machine Learning regression algorithms. The dataset contains various housing features such as area, number of bedrooms, bathrooms, parking spaces, furnishing status, and other amenities.

The objective is to build and compare multiple regression models to identify the best-performing algorithm for house price prediction.

## 📂 Dataset

- Dataset Name: Housing.csv
- Total Records: 545
- Total Features: 13

### Features

- Area
- Bedrooms
- Bathrooms
- Stories
- Main Road
- Guest Room
- Basement
- Hot Water Heating
- Air Conditioning
- Parking
- Preferred Area
- Furnishing Status

**Target Variable**
- Price

## 📈 Evaluation Metrics

The models were evaluated using:

- R² Score

- Mean Squared Error (MSE)
- Root Mean Squared Error (RMSE)

## 🎯 Results

The performance of all four regression models was compared, and the best-performing model was selected based on the evaluation metrics.
